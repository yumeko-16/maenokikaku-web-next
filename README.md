# Maeno Planning Co., Ltd. Official Website (Next.js Renewal)

This repository is a new frontend project that has been redesigned using Next.js for the existing corporate official website, "Maeno Planning Co., Ltd. Official Website."

## Technologies Used

| Category        | Technology                                     |
| --------------- | ---------------------------------------------- |
| Framework       | Next.js                                        |
| Language        | JavaScript                                     |
| Styling         | SCSS / CSS Modules                             |
| Build / Hosting | Vercel                                         |
| Other           | Node.js / npm                                  |

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yumeko-16/maenokikaku-web-next.git
cd maenokikaku-web-next
```

### 2. Install dependencies

```bash
npm install
```

### 3. Set environment variables

Create a .env.local file in the project root and add the required environment variables:

```bash
SENDGRID_API_KEY=xxxxx
```

### 4. Start the development server

```bash
npm run dev
```

The site will be available at:

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
